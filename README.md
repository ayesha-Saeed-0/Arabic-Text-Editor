Arabic Text Editor 
A Java desktop application for composing and editing Arabic text, built with full right-to-left (RTL) support using the Eclipse IDE.


Features

RTL-first editing — text area is configured for right-to-left Arabic script
Arabic Unicode support — handles the full Arabic character set correctly
Desktop GUI — built with Java Swing/AWT for a native-feeling window
Lightweight — no external dependencies; runs on any platform with a JRE

Project Structure


Arabic-Text-Editor/
├── src/               # Java source files
├── bin/               # Compiled .class files
├── .settings/         # Eclipse IDE settings
├── .classpath         # Eclipse classpath configuration
└── .project           # Eclipse project descriptor (project name: TextEditor)

Requirements

- Java 8 or higher (JDK for building, JRE for running)
- Eclipse IDE (optional — for opening/editing the project directly)

Getting Started

Option 1 — Run from Eclipse

1. Clone the repository:
   
   git clone https://github.com/ayesha-Saeed-0/Arabic-Text-Editor.git
   
2. Open Eclipse and go to File → Import → Existing Projects into Workspace.
3. Select the cloned folder and click Finish.
4. Right-click the project → Run As → Java Application.

Option 2 — Compile & Run from the command line

Clone
git clone https://github.com/ayesha-Saeed-0/Arabic-Text-Editor.git
cd Arabic-Text-Editor

 Usage

1. Launch the application.
2. Type or paste Arabic text directly into the editor area.
3. Text flows right-to-left automatically.
